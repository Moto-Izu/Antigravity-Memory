## 2. Infrastructure Insight (For Jeff Dean)
- **Install BigQuery/PostgreSQL MCP**: Directly query database schemas to avoid performance bottlenecks.
- **Install Cloud Run/AWS MCP**: Monitor system metrics to predict 10^9 scale loads.

## 3. Low-Level Execution (For Fabrice Bellard)
- **Install Docker MCP**: Create minimalist development environments on the fly.
- **Install Shell MCP**: Direct system-level optimization without layer overhead.
